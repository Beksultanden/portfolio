Development of a VR Application for PC Assembly

This project is an educational Game in which the user can virtual assembly a personal computer using VR glasses and a controller. This is especially useful for beginners who want to build a PC with their own hands. After launching the application, the user enters a virtual room with a laptop. With the help of a VR controller, he can move around the room, interact with objects, select components and assembly Pc step by step. there is a laptop in the room and there you can select the necessary parts, processor, graphics card, RAM and other components. After the selection, all the items appear on the desktop. The user can take them with virtual hands (via the controller) and assemble the PC by following the instructions on the screen. After the build is completed, the virtual computer "turns on" and the game ends. Technologies used: Blender – for modeling 3D objects (graphics card, processor, case, etc.). Unity - for building the interactive environment. C# - for implementing logic and interaction. VR Headset (connected to a smartphone) - for user control and immersion.

Image coloring based on deep learning.

The Master's thesis project is dedicated to the task of automatic coloring of black and white images using deep learning methods. The main idea of the project is to develop and train a neural network model capable of reconstructing color information based on the structure and content of a black-and-white photograph. The input for the model is a grayscale image, a black-and—white photo, and the output is a colorized version of it. To train the model, the COCO-Stuff dataset was selected, which consists of 116,000 color photographs of various resolutions. Before sending all the images to the model, they were reduced to a single size of 256×256 pixels, and then divided into training and test samples. The model was trained in the Jupyter Notebook environment using Python and libraries such as PyTorch, NumPy, OpenCV, Matplotlib and other machine learning and computer vision tools. In the process, the architecture of the U-Net type model was implemented, which consists of an encoder and a decoder: the encoder consists of 7 blocks, and the decoder consists of 3 blocks. Additional improvements were also added to the model, including attention mechanisms (CBAM), and quality assessment metrics were used for comparison.: PSNR, SSIM, LPIPS. After training, the model was tested on several datasets to evaluate its versatility and color accuracy. The model showed good results by successfully coloring images that were not used in training.
